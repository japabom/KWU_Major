README

1. 프로그램 설명
 1) 정해진 자본을 이용하여 사용자가 원하는 구단을 선택하고 
    원하는 선수를 영입/방출 하여 나만의 구단을 만드는 프로그램

 2) 오락실에서 한번씩은 봤을 에어 하키 게임을 1P, 2P 두 명이서 할 수 있다.

2. 프로그램 실행방법
 1) 게임 시작 버튼 - (1) 에어 하키 게임 실행 하얀 화면이 나타나면 SPACE 키를 눌러서 시작

		(2) 1P 이동 키 : a=왼쪽 이동, s=아래 이동, d=오른쪽 이동, w= 위쪽 이동
		
		(3) 2P 이동 키 : 화살표 키

		(4) 게임 시작 후 ESC 키 누르면 게임 종료

 2) 구단 만들기 - (1) 사용자가 원하는 구단 버튼 클릭

	         (2) 영입 버튼 -> 영입을 원하는 포지션 선택 -> 선수 선택 -> 영입 버튼 누르면 영입 완료 -> 완료 버튼 누르면 창 닫힘
		영입 버튼을 눌렀을 때 남아 있는 자본이 부족하거나 이미 선택한 선수가 구단에 있으면 영입 실패
	         
	         (3) 방출 버튼 -> 방출을 원하는 포지션 선택 -> 선수 선택 -> 방출 버튼 누르면 방출 완료 -> 완료 버튼 누르면 창 닫힘
		방출 버튼을 눌렀을 때  선택한 선수가 구단에 없을 때 방출 실패

	         (4) 구단 현황 - 현재 구단에 있는 선수, 남아 있는 자본을 Shell에서 알려줌
 
 3) 구단 초기화 - 사용자가 초기화를 원하는 구단 버튼을 클릭하면 초기화 완료

 4) 종료 - 프로그램 종료

2. 필요한 패키지 
 Pillow, Pygame 패키지 설치 필요

3. 수정하지 못한 오류
  (1) 게임에서 1P와 2P가 키를 같이 누르면 한쪽에서 키가 중복으로 입력된 것으로 인식되어 이동이 2번씩 됨

  (2) 게임에서 공이 골인했을 때 SCORE를 나타내는 메세지를 표현하려고 시도 했으나 실패함

  (3) 게임에서 시작 화면에 ‘Press SPACE key to start’ 라는 메세지를 출력하고 싶었으나 실패함 
  
  (4) 게임에서 플레이어 이미지와 공 이미지의 충돌 처리 과정에서 이미지가 겹쳤을 때 끼임 현상이 발생함

  (5) 영입 시 선택한 선수가 이미 구단에 있거나, 방출 시 선택한 선수가 없을 시 오류 메세지 출력 후 영입/방출 완료 메시지가 출력됨

  (6) 자식 윈도우 창에서 버튼을 누를 때마다 그 전 윈도우 창이 앞으로 올라와서 현재 작업 중인 창이 뒤로 밀려남
